### ✅ 验证清单

- [ ] `/pm` 命令能正确加载产品经理角色
- [ ] `/ui` 或 `/designer` 能加载设计师角色
- [ ] `/fe` 或 `/frontend` 能加载前端工程师角色
- [ ] `/be` 或 `/backend` 能加载后端工程师角色
- [ ] 各Agent输出符合其prompt定义的规范
- [ ] 角色切换时上下文正确保持
- [ ] 团队协作模式能协调多个Agent
